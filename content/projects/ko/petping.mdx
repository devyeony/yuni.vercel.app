## 📌 프로젝트명
- **펫핑 (Petping)**

## 📋 프로젝트 개요
- <b>펫핑 (Petping)</b>은 **반려동물**의 **산책**, **영양**, **쇼핑**, **보험**을 **통합 관리**할 수 있는 **올인원 헬스케어 앱**입니다.
- 사용자와 반려동물의 **건강한 라이프스타일**을 지원하기 위한 **다양한 기능**을 제공하며, 앱 전반의 **사용자 경험**과 **서비스 운영 효율성**을 지속적으로 개선해 왔습니다.

## 📅 기간
- 2021년 7월 – 2024년 1월

## 🛠 기술 스택
- **백엔드:** Java, Spring, Spring Boot, JPA, MyBatis
- **프론트엔드:** JavaScript, HTML, CSS
- **데이터베이스:** MySQL, PostgreSQL, Redis
- **DevOps:** Docker, Jenkins, Linux
- **클라우드 인프라 (AWS 서비스):** EC2, S3, ECS, ElastiCache, CodeDeploy, IAM, Route 53
- **모니터링 & 로깅:** Grafana, Loki
- **데이터 시각화 & BI 도구:** Redash

## ✨ 주요 기능
| **기능** | **설명** |
|-------------|---------|
| **펫 케어 대시보드** | 펫 건강 관련 데이터 요약, 날씨 및 미세먼지 정보 제공 |
| **펫 프로필** | 펫 프로필 설정, 미션 펫 설정, 가족코드 공유 |
| **산책 기능** | 소변·대변·마킹 기록, 오디오 가이드 산책, 산책 경로 저장, 위치 기반 산책 친구 찾기 |
| **영양 기능** | 사료 검색, 식사 기록, 칼로리 분석, 반복식단 관리, 알러지 관리 |
| **건강 캘린더** | 산책·식단 통합 캘린더, 건강 다이어리 관리, 건강 수치 기록, 건강 기록 소셜 공유 |
| **미션 및 리워드** | 출석·산책·이벤트 미션, 포인트 적립·사용·만료 |
| **푸시 및 알림 센터** | 유저 활동 기반 자동 푸시, 미션 진행·완료 알림, 앱 내 인앱 알림 제공 |
| **커머스 기능** | 쇼핑몰 연동 및 포인트 사용, 추천 상품 및 유저 맞춤형 상품 배너 노출 |
| **펫보험 기능** | 펫보험 계약·갱신·청구, 보험사 연동, 펫보험 리워드 통합 |
