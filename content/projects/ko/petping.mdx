## 📌 프로젝트명
- **펫핑 (Petping)**

## 📋 프로젝트 개요
- **펫핑**은 반려동물의 **산책, 식사, 건강, 보험, 쇼핑**을 통합 관리할 수 있는 **헬스케어 앱**입니다.  
- 사용자와 반려동물의 **건강한 라이프스타일**을 지원하며, 실시간 알림, 보험 청구 자동화, 커머스 연동 등 다양한 기능을 제공합니다.


## 📅 기간
- 2021년 7월 – 2024년 1월

## 🛠 기술 스택
- **백엔드:** Java, Spring, Spring Boot, JPA, MyBatis
- **프론트엔드:** JavaScript, HTML, CSS
- **데이터베이스:** MySQL, PostgreSQL, Redis
- **DevOps:** Docker, Jenkins, Linux
- **클라우드 인프라 (AWS 서비스):** EC2, S3, ECS, ElastiCache, CodeDeploy, IAM, Route 53
- **모니터링 & 로깅:** Grafana, Loki
- **데이터 시각화 & BI 도구:** Redash

## 👤 주요 역할
- 백엔드 아키텍처 설계 및 API 개발 전반 주도  
- 서비스 확장성과 안정성을 고려한 인프라 설계 및 운영  
- PM, 디자이너, 앱/프론트 개발자, QA, 데이터 분석가 등 다양한 직군과 협업  
- 실시간 운영 대응, CS 자동화, 배포 파이프라인 구축 등 전사적 운영 효율화에 기여  
- 사용자 경험 개선을 위한 기능 고도화 및 품질 안정화 주도

## ✨ 핵심 기여 및 성과  

- **산책 이벤트 및 푸시 자동화 기능 개발**  
  → 📈 MAU **30% 증가**, 📊 DAU **20% 상승**

- **펫보험 청구 프로세스 자동화 및 외부 커머스 API 연동**  
  → 💬 보험 관련 CS 문의 **20% 감소**, 🤝 커머스 연동 품질 **90% 달성**

- **CI/CD 자동화 및 블루/그린 무중단 배포 구축**  
  → 🚀 배포 오류 감소, 무중단 배포율 **95% 달성**

- **Grafana·Loki 기반 실시간 로그 모니터링 체계 구축**  
  → 🔎 장애 탐지 및 대응 시간 **60% 단축**, 서비스 안정성 강화

- **Redash 기반 데이터 조회 환경 개선**  
  → ⚙️ 운영팀의 일별 리포트 및 통계 업무 자동화로 **데이터 처리 시간 50% 단축**

- **QA 전용 테스트 API 및 쿼리 제공**  
  → ✅ 테스트 커버리지 **40% 증가**, QA의 자율 테스트 환경 확보

- **Slack·Jira·Confluence 자동화**  
  → 🔔 이슈 공유 속도 향상, 커뮤니케이션 지연 **70% 감소**

- **JPA 도입 및 MyBatis 혼용 구조 정착**  
  → 코드 간결화 및 유지보수성 개선, 신규 기능 개발 속도 향상
