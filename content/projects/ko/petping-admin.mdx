## 📌 프로젝트명
- **펫핑 어드민 (Petping Admin)**

## 📋 프로젝트 개요
- <b>‘펫핑 어드민 (Petping Admin)’</b>은 <a href="/ko/projects/petping" target="_blank"><b>펫핑</b></a> 서비스의 **내부 운영**과 **고객 응대**를 지원하는 **백오피스 시스템**으로, **콘텐츠, 사용자, 리워드, 이벤트** 등을 효율적으로 관리할 수 있도록 개발되었습니다. 
- 운영팀의 **업무 효율성**과 **서비스 안정성** 향상에 중점을 두었습니다.

## 📅 기간
- 2021년 7월 – 2024년 1월

## 🛠 기술 스택
- **백엔드:** Java, Spring Boot, Spring Security, JPA, MyBatis
- **프론트엔드:** JavaScript, HTML, CSS
- **템플릿 엔진:** Thymeleaf
- **데이터베이스:** PostgreSQL
- **UI 프레임워크 및 라이브러리:** Bootstrap, DataTables.js, Chart.js

## 👤 주요 역할
- 전체 시스템 아키텍처 설계 및 풀스택 개발 주도  
- 운영팀 요구사항 분석 및 비즈니스 로직 설계  
- 반복 업무 자동화, UI/UX 개선, 보안 강화 기능 직접 구현  
- 운영 효율성과 고객 응대 품질 향상에 집중

## ✨ 핵심 기여 및 성과

- 🔄 **반복 업무 자동화 기능 개발**  
  → 운영 업무 처리량 **2배** 증가, 수동 처리 시간 대폭 감소

- 📊 **사용자별 API 호출 로그 뷰어 구현**  
  → 고객 응대 속도 **40%** 단축, 문제 해결 신속화

- 🔐 **Google OAuth2 및 OTP 2단계 인증 도입**  
  → 관리자 계정 보안 강화, 무단 접근 위험 최소화

- 🗺️ **네이버 지도 API 연동 산책 데이터 시각화**  
  → 운영 대응 시간 **25%** 감소, 이벤트 모니터링 효율 향상

- ✉️ **펫보험 청구 이메일 재발송 및 수신자 주소 수정 기능 추가**  
  → 이메일 오류 대응 독립성 확보, 운영 신뢰성 강화
